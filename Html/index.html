VALORANT is a Riot game created by Riot Games is a free multiplayer first person shooter game developed by. The game was first released in October 2019 under the codename 'Project' A》， A closed beta version was released on April 7, 2020. On June 2, 2020, Valorant began public testing. On September 4, 2021, Tencent announced the introduction of VALORANT. In December 2022, 'Fearless Contract' was selected as a WCG competition project.

The game is a 5V5 game centered around hero characters The tactical shooting online game adopts the classic FPS "explosive mode" as the core gameplay, where players will transform into tactical heroes with different skills and participate in battles using different types of weapons and firearms.
